How to edit:
1. Open the folder in VS Code
2. Edit text in any HTML page
3. Replace or add photos in the images folder
4. Change styling in css/style.css
5. Update gallery behavior in js/site.js

How to add another image:
1. Put the new photo in images/
2. Open the page where it belongs
3. Copy one gallery-card block and change the image file name and caption

Note:
This is a front-end site only. The contact section is informational and does not submit forms online.
